# Скрипты для быстрой проверки лабораторной работы

Эта папка содержит скрипты для автоматизации настройки и проверки проекта.

## Основные скрипты

### setup.bat
Полная автоматическая настройка проекта:
- Установка зависимостей
- Запуск MinIO
- Инициализация DVC
- Настройка remote
- Генерация файла user_history.csv (если его нет)
- Добавление данных в DVC (версия v1.0 - октябрь)

**Использование:**
```bash
scripts\setup.bat
```

### prepare_data.bat
Подготовка данных для DVC:
- Добавление user_history.csv в DVC (версия v1.0 - октябрь)
- Отправка данных в MinIO

**Использование:**
```bash
scripts\prepare_data.bat
```

### demo.bat
Демонстрация работы системы:
- Проверка MinIO
- Проверка DVC конфигурации
- Запуск скрипта инициализации данных

**Использование:**
```bash
scripts\demo.bat
```

### add_november_data.bat
Добавление данных за ноябрь и создание версии v2.0:
- Добавление новых данных о просмотрах за ноябрь
- Обновление версии в DVC (v2.0 - октябрь + ноябрь)
- Отправка в MinIO

**Использование:**
```bash
scripts\add_november_data.bat
```

### switch_to_october.bat
Переключение на версию v1.0 (только октябрь):
- Откат к версии с данными только за октябрь
- Просмотр текущей версии

**Использование:**
```bash
scripts\switch_to_october.bat
```

### switch_to_november.bat
Переключение на версию v2.0 (октябрь + ноябрь):
- Переключение на версию с данными за октябрь и ноябрь
- Просмотр текущей версии

**Использование:**
```bash
scripts\switch_to_november.bat
```

### clean_and_reset.bat
Полная очистка и перезапуск проекта:
- Остановка MinIO
- Удаление локальных данных
- Перезапуск с нуля

**Использование:**
```bash
scripts\clean_and_reset.bat
```

### generate_user_history.py / generate_user_history.bat
Генерация файла user_history.csv с историей взаимодействий пользователей с товарами:
- Настройка количества строк
- Настройка seed для воспроизводимости
- Автоматическое создание папки data, если её нет
- Генерирует числовые значения для user_id (1-20) и item_id (1-50)

**Важно**: После настройки DVC необходимо сгенерировать файл перед добавлением его в версионный контроль.

**Использование:**
```bash
# Через bat-файл (интерактивный режим)
scripts\generate_user_history.bat

# Через bat-файл с параметрами
scripts\generate_user_history.bat 100 42

# Напрямую через Python
poetry run python scripts/generate_user_history.py --rows 100 --seed 42
poetry run python scripts/generate_user_history.py --rows 50
poetry run python scripts/generate_user_history.py  # 100 строк, случайный seed
```

**Параметры:**
- `--rows` - количество строк для генерации (по умолчанию: 100)
- `--seed` - seed для генератора случайных чисел (для воспроизводимости)
- `--output` - путь к выходному файлу (по умолчанию: data/user_history.csv)

**Формат данных:**
Файл содержит колонки `user_id` и `item_id` с числовыми значениями:
```csv
user_id,item_id
1,5
1,12
2,3
...
```

## Последовательность проверки

1. **Первичная настройка:**
   ```bash
   scripts\setup.bat
   ```
   Скрипт автоматически сгенерирует файл user_history.csv, если его нет.

2. **Демонстрация работы:**
   ```bash
   scripts\demo.bat
   ```

3. **Создание версии v2.0 (добавление данных за ноябрь):**
   ```bash
   scripts\add_november_data.bat
   ```

4. **Переключение между версиями:**
   ```bash
   scripts\switch_to_october.bat    # Версия v1.0 (октябрь)
   scripts\switch_to_november.bat   # Версия v2.0 (октябрь + ноябрь)
   ```

## Ручная настройка (если нужно)

Если вы настраиваете проект вручную, после настройки DVC remote необходимо:

1. **Сгенерировать файл user_history.csv:**
   ```bash
   poetry run python scripts/generate_user_history.py --rows 100 --seed 42
   ```

2. **Добавить файл в DVC:**
   ```bash
   dvc add data/user_history.csv
   dvc push
   ```

## Версии данных

- **v1.0 (октябрь)**: Только данные за октябрь
- **v2.0 (октябрь + ноябрь)**: Данные за октябрь и ноябрь
