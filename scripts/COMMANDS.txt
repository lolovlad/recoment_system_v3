========================================
КОМАНДЫ ДЛЯ БЫСТРОЙ ПРОВЕРКИ ЛАБОРАТОРНОЙ РАБОТЫ №2
========================================

1. ПОЛНАЯ АВТОМАТИЧЕСКАЯ НАСТРОЙКА:
   scripts\setup.bat

2. ПРОВЕРКА РАБОТЫ СИСТЕМЫ:
   scripts\demo.bat

3. СОЗДАНИЕ ВЕРСИИ v2.0 (ДОБАВЛЕНИЕ ДАННЫХ ЗА НОЯБРЬ):
   scripts\add_november_data.bat

4. ПЕРЕКЛЮЧЕНИЕ МЕЖДУ ВЕРСИЯМИ:
   scripts\switch_to_october.bat    # Версия v1.0 (октябрь)
   scripts\switch_to_november.bat     # Версия v2.0 (октябрь + ноябрь)

5. ГЕНЕРАЦИЯ ФАЙЛА user_history.csv:
   scripts\generate_user_history.bat [количество_строк] [seed]
   
   Примеры:
   scripts\generate_user_history.bat 100 42
   scripts\generate_user_history.bat 50
   scripts\generate_user_history.bat  # Интерактивный режим

========================================
РУЧНЫЕ КОМАНДЫ (если нужны):
========================================

Генерация user_history.csv:
   poetry run python scripts/generate_user_history.py --rows 100 --seed 42
   poetry run python scripts/generate_user_history.py --rows 50
   poetry run python scripts/generate_user_history.py  # 100 строк, случайный seed

Установка зависимостей:
   poetry install

Запуск MinIO:
   docker-compose up -d

Остановка MinIO:
   docker-compose down

Проверка статуса MinIO:
   docker-compose ps

Инициализация DVC (если нужно):
   dvc init

Настройка DVC remote:
   dvc remote add -d myremote s3://datasets
   dvc remote modify myremote endpointurl http://localhost:9000
   dvc remote modify myremote access_key_id minioadmin
   dvc remote modify myremote secret_access_key minioadmin

Добавление данных в DVC (версия v1.0 - октябрь):
   dvc add data/user_history.csv

Отправка данных в MinIO:
   dvc push

Скачивание данных из MinIO:
   dvc pull

Инициализация данных в приложении:
   poetry run python scripts/init_data.py

Просмотр DVC конфигурации:
   dvc remote list

Просмотр версионированных файлов:
   dvc list data/

Переключение между версиями:
   dvc checkout data/user_history.csv.dvc

========================================
ПРОВЕРКА В MINIO CONSOLE:
========================================

1. Откройте браузер: http://localhost:9001
2. Войдите: minioadmin / minioadmin
3. Перейдите в бакет "datasets"
4. Убедитесь, что файл user_history.csv присутствует
